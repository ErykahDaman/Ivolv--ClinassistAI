# ClinAssist AI

**AI-Powered Electronic Medical Records System for Botswana**

ClinAssist AI is a voice-enabled, intelligent Electronic Medical Record (EMR) platform developed by **Ivolv (Pty) Ltd**. It helps doctors and clinics in Botswana reduce paperwork by automatically converting doctor-patient conversations into structured clinical notes.

**Tagline:** Digital Healthcare Solutions for Botswana

---

## ✨ Features

- Real-time Voice-to-Note Transcription
- Automated Clinical Documentation
- Smart Record Search & Retrieval
- AI-Powered Prescription Suggestions
- Secure Cloud Storage
- Role-based Access (Doctor & Patient)
- Fully compliant with Botswana Data Protection Act & Cybersecurity Act 2025
